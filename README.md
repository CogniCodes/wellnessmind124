# 🌿 WellnessMind (SereneMind)

WellnessMind is an AI-powered mental wellness platform designed to help users monitor their emotional well-being, track symptoms, and gain personalized wellness insights. The platform encourages proactive healthcare by identifying emotional barriers that often prevent people from seeking medical help.

🔗 Live Website: https://wellnessmind124.vercel.app

---

## ✨ Features

- 🤖 AI Wellness Chat Assistant
- 😊 Daily Mood Tracking
- 🩺 Symptom Logging
- 📈 Mood & Symptom History
- 💡 Personalized AI Wellness Suggestions
- 👥 Community Support Space
- ❤️ Support Circle Management
- 👤 User Profile Management
- 📊 Interactive Health Dashboard
- 🔔 Notifications

---

## 🎯 Problem Statement

Many individuals delay seeking healthcare due to:

- Fear of diagnosis
- Social stigma
- Masculinity norms
- Cost concerns
- Lack of time
- Denial and uncertainty

WellnessMind addresses these challenges through AI-guided conversations, symptom tracking, and personalized wellness recommendations.

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/CogniCodes/wellnessmind124.git
```

### Install dependencies

```bash
npm install
```

### Start development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```
